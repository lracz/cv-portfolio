# Cinematic Portfolio & CV Site

A premium, production-grade professional portfolio built with **React**, **Vite**, and **Vanilla CSS**. Designed with a "Minimalist Luxury" aesthetic, featuring dark mode, bilingual support (HU/EN), and AI-generated project mockups.

![Portfolio Preview](public/projects/hero-preview.png) *(Note: Add a hero preview image here later)*

## 🚀 Key Features

- **Dark/Light Mode**: Seamless theme switching with persistent state.
- **Bilingual Interface**: Full Hungarian and English localization.
- **Interactive UI**: 
  - Scroll progress tracking.
  - Smooth reveal animations.
  - Filterable projects gallery.
- **AI-Enhanced**: Mockups and visuals generated using advanced AI models.
- **SEO Optimized**: Complete OpenGraph and Twitter card metadata.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite 8.
- **Styling**: Vanilla CSS (Custom Design System).
- **Icons**: Lucide-style SVG icons.
- **Deployment**: Optimized for Vercel/Netlify.

## 📦 Project Structure

```text
src/
├── components/   # Modular UI components (Navbar, Hero, Projects, etc.)
├── data/         # Localization and project data
├── hooks/        # Custom React hooks (useTheme, useLanguage)
└── styles/       # Atomic CSS design system
```
